This plugin integrates [YVanish](https://github.com/Ynfuien/YVanish) with [DiscordSRV](https://github.com/DiscordSRV/DiscordSRV) to show the missing fake join and leave messages in chosen the server chat channel in your Discord server. No setup is nessescary, just drop the jar into your plugins folder and start your server! :D
